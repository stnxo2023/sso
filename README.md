# sso

Thanks for all the interest this project has had over the years. We no longer maintain this public repository. You are still very welcome to build from source, or keep your own forks going; the code remains available in git history. If you are evaluating options for a new deployment, we encourage you to look for more modern alternatives in this space, or active forks of **sso** such as [Pomerium](https://github.com/pomerium/pomerium).
